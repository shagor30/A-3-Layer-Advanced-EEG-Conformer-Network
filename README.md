# A-3-Layer-Advanced-EEG-Conformer-Network
Official repository for the 3-Layer Advanced EEG Conformer framework developed at CUET[cite: 1, 2, 3]. It replaces heavy multi-branch networks with sequential spatiotemporal convolutions and a serialized 3-layer attention cascade[cite: 28, 29, 46]. Achieved state-of-the-art accuracies of 88.33% on Dataset 2a and 92.46% on Dataset 2b[cite: 222, 225]
